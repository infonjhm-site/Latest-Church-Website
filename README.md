# New Jerusalem Harvest Ministries Website

This project is a modern, responsive multi-page church website designed to establish a strong online presence for **New Jerusalem Harvest Ministries**. It provides visitors with clear and structured access to church information, ministry departments, events, online participation options, giving resources, and account-related services in an organized and mobile-friendly experience.

## 🌟 Project Overview

The website is built as a **static front-end application** with well-structured page sections and interactive user interface elements. The goal of the platform is to support church communication and engagement by helping visitors easily:

* Learn about the church, its mission, and spiritual leadership
* Explore various ministry departments and opportunities to serve
* Discover upcoming events and join online services
* Access account pages for registration and login
* Submit contact or support requests
* Contribute through online giving resources

---

## ✨ Key Features

* Responsive navigation with dropdown menus for all major site sections
* Fullscreen **video hero section** on the homepage
* Featured worship **video carousel** with keyboard and swipe support
* Upcoming events section with engagement-focused call-to-action areas
* Scroll progress indicator and **sticky navigation bar**
* Floating **back-to-top button** for improved user navigation
* Animated counters and lightweight interaction effects
* Client-side form handling for contact, login, registration, and support forms

## 🚀 Running the Project Locally

Since this is a **static front-end project**, it can be easily run using a local development server.

### Step 1
Open the project folder in **VS Code** (or your preferred editor).

### Step 2
Start a local server using Python (or Live Server extension):

```bash
python3 -m http.server 5500
```

### Step 3
Open your browser and navigate to:

```text
http://localhost:5500
```

The website will now be running locally.

## 🌐 Deployment Instructions

You can easily deploy this static site for free using platforms like **GitHub Pages**, **Netlify**, or **Vercel**.

### GitHub Pages
1. Push your code to a GitHub repository.
2. Go to **Settings** > **Pages** in your repository.
3. Under **Build and deployment**, select `Deploy from a branch`.
4. Choose the `main` or `master` branch and save.
5. Your site will be live within a few minutes!

### Netlify / Vercel
1. Sign up for [Netlify](https://www.netlify.com/) or [Vercel](https://vercel.com/).
2. Connect your GitHub account and import this project's repository.
3. Leave the build command empty (since it's a static site) and set the publish directory to the root folder.
4. Click **Deploy**.
